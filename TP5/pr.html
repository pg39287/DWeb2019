ABSTRACT:

Trabalho de casa 5 consistiu na criação de um website para gestão de tarefas, onde são listadas e criadas tarefas. A criação do mesmo foi feita em servidor nodejs com outras tecnologias de apoio.

O website consiste numa página singular de HTML em que são apresentadas as tarefas criadas numa tabela. Existem dois botões, o botão de adicionar tarefa e o botão de eliminar tarefa, ao lado de cada uma representada pelo icone do lixo.

Organização do projeto encontra-se disposta em a pasta base com os scripts e ficheiros de configuração: server.js onde é iniciado o servidor e gerido o roteamento, handler.js responsável pelas funções de apoio e controlo às acções e funcionalidades do servidor e serverConfig.json onde estão as configurações do servidor guardadas e que são lidas ao iniciar. Na pasta database está o ficheiro de persistência de dados tasks.json, em que cada tarefa é composta por um id, título, descrição e data. Na pasta www estão as views e favicon, em que layout.pug é a base do documento HTML com as configurações e imports, index.pug é o conteúdo do body que é a página com os botões e tabela de tarefas e error.pug para mostrar uma página de erro quando se tenta aceder a uma rota que não existe.

Foram usadas as seguintes tecnologias para suporte na criação deste website: pugjs para templates das views, bootstrap 4 para o design e estilo das páginas, bootstrap-validate.js para validar os elementos do form, nodemon para reiniciar o servidor a cada mudança feita no projeto e material-design-icons. para os icones na lista de tarefas.
