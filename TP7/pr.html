ABSTRACT:

Trabalho de casa 7 está dividido em duas partes: - 3 Exercícios de queries MongoDB - Website para gestão de canções

O primeiro trabalho de casa consiste em 3 queries dadas na aula para resolver como exercícios complementares para aprender a trabalhar com MongoDB. As queries eram 1.) Listar gêneros de filmes e indicar quantos há de cada um, 2.) Quais os títulos dos filmes do gênero "Drama" e Que atores participaram em filmes do gênero "Comedy".

O segundo trabalho de casa consiste num website como os passados trabalhos de casa, em que gere filmes numa base de dados em mongo. O website está separado em duas aplicações, o website com gestão e frontend para pesquisar filmes pelo título e para adicionar, editar ou apagar filmes da base de dados e a segunda é uma REST API que interage com base de dados separado em várias rotas e pedidos.

O servidor possuí um controlador movieController.js responsável pela interação com a base de dados, e 3 routers, o indexRouter.js responsável pela homepage na caminho /, o movieRouter.js responsável pelas rotas /movies para vários pedidos relacionados com os filmes e o apiRouter.js responsável pela rota /api-movies que contém os pedidos à API do website para gerir pedidos HTTP para recuperar informação em json.

Foram usadas as seguintes tecnologias para suporte na criação deste website: pugjs para templates das views, bootstrap 4 para o design e estilo das páginas, nodemon para reiniciar o servidor a cada mudança feita no projeto, axios para apoio nos pedidos HTTP no lado do cliente e mongoose js ORM para interagir com o mongo.

Deliverables:

  1. Exercícios de queries MongoDB
  2. Instruções de importação da base de dados MongoDB
  3. app.js: Ficheiro do servidor

  5. movieController.js: Controller dos filmes
  6. indexRouter: router para o index do website

  8. apiRouter: router dos pedidos API
