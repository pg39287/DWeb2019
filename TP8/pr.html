ABSTRACT:

Trabalho de casa 8 consiste numa REST API para extrair informação sobre prémios nobel e os seus laureados. O servidor possuí um controlador prizeController.js responsável pela interação com a base de dados e o router apiRouter.js responsável pelas à API do website para gerir pedidos HTTP para recuperar informação em json.

Foram usadas as seguintes tecnologias para suporte na criação desta API: nodemon para reiniciar o servidor a cada mudança feita no projeto e mongoose js ORM para interagir com o mongo.

Deliverables:

  1. README para as rotas e o exercício do TPC com as instruções necessárias para o funcionamento
  2. app.js: Ficheiro do servidor
  3. prize.js: Schema e Modelo de um prémio mais laureados na base de dados
  4. apiResponse.js: Estrutura de resposta da API
  5. prizeController.js: Controller dos prémios para as queries na base de dados
  6. apiRouter: router para os diferentes endpoints da API
