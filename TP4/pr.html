ABSTRACT:

Trabalho de casa 4 consistiu na criação de um servidor nodejs para um website dos arqueossítios do TP3. Com apoio do arq.xml é retirado o conteúdo para apresentar em cada elemento.

Segunda parte foi criar no arq.xls do TP3 e adaptar à geração das páginas e conteúdos para os arqueossítios. Foi adicionado no template do ARQELEM do ficheiro uma instrução de processamento para o template que deve ser assumido para a transformação nomeadamente o template.xls. Foi movido o template das páginas individuais para um só ficheiro template.xls que é responsável pela transformação dos xmls individuais dos arqueossítios.

Deliverables:

  1. server.js
  2. arq.xsl
