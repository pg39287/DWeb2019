ABSTRACT:

Trabalho de casa 6 consistiu na criação de um website para gestão de canções, onde são listadas e geridas canções tradicionais. A criação do mesmo foi feita em servidor nodejs com outras tecnologias de apoio.

O website consiste num website em que apresenta um menu no topo com vários locais e um botão na homepage para adicionar uma canção nova. Existe no topo a barra de menu onde se pode seleccionar a lista das canções todas, ou lista por região as canções. Na tabela das canções podemos ainda editar e apagar a canção da linha correspondente.

O projeto foi gerado com o comando do expressjs para criar um projeto raíz já com vários ficheiros gerados para apoio à criação do website. De notar os ficheiros mais importantes são app.js de onde o servidor se inicia, index.js onde as rotas do website estão centradas, database.json onde as canções são guardadas e as variadas views do websites.

Foram usadas as seguintes tecnologias para suporte na criação deste website: pugjs para templates das views, bootstrap 4 para o design e estilo das páginas, nodemon para reiniciar o servidor a cada mudança feita no projeto e axios para apoio nos pedidos HTTP no lado do cliente, nanoid para geração de IDs para as canções.
